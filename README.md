# Formula1
Nettisivu, joka hakee tietokannasta tuloksia ja muuta tietoa Formula1 -sarjan kauldelta 2022. SQL, PHP, HTML, CSS

<img width="507" height="340" alt="image" src="https://github.com/user-attachments/assets/d64201b1-48e5-4ac4-bd82-0cb89a8dd504" />

Sivustolla on oma osionsa kauden ajajille, talleille, sponsoreille, radoille, osakilpailuille ja tuloksille sekä alasivu jossa näytetään edeltävien kausien voittajat. 

Tietokannasta löytyy taulukot kausien voittajille, radoille, ajajille, sille mihin osakilpailuihin ajaja osallistui, osakilpailuille, tuloksille, sponsoreille ja talleille

Käytetyt teknologia

PHP – tietokantayhteys ja datan käsittely <br>
MySQL – tietojen tallennus <br>
HTML – sivun rakenne <br>
CSS – ulkoasu ja taulukon tyylittely <br>
Kehitysympäristö: XAMPP + phpMyAdmin <br>


<br>Käyttöönotto paikallisesti

Asenna XAMPP ja käynnistä Apache ja MySQL<br>
Kloonaa tämä repositorio htdocs-kansioon<br>
Avaa phpMyAdmin ja luo uusi tietokanta<br>
Tuo f1v3.sql-tiedosto luotuun tietokantaan<br>
Muokkaa database.php-tiedostoon tietokannan tunnukset<br>
Avaa sivu selaimessa osoitteessa http: //localhost/projektin-kansio
